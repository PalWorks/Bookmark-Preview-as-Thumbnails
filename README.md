# BookmarksThumbnails

A Chrome extension to preview bookmarks as thumbnails.

## Features
- Capture thumbnails of bookmarks.
- Store thumbnails locally using IndexedDB.
- Optional storage to local file system.
- Safari-style Tab Overview interface.

## Development

### Install dependencies
```bash
npm install
```

### Run dev server
```bash
npm run dev
```

### Build
```bash
npm run build
```

### Test
```bash
npm test
```
